HTML 1. HTML的骨架结构

HTML(超文本标记语言)是网页的基础骨架，它定义了网页的结构和内容。

HTML 2. 文本结构标签

在HTML中，文本结构标签用于组织和结构化文本内容，使内容更具语义化，便于搜索引擎理解和页面可访问性。

HTML 3. 主要媒体与嵌入标签

HTML中的主要媒体与嵌入标签用于在网页中嵌入外部资源，如图像、音频、视频、其他网页或应用程序，以丰富页面内容与交互。

HTML 4. 基本表格创建

表格是 HTML 中用于展示结构化数据的重要元素，通过行和列的网格来组织信息。

HTML 5. 表单与用户交互

表单是网页中收集用户数据并提交给服务器处理的交互工具。

HTML 6. 分区与语义化布局

HTML5的语义化布局元素让网页结构更加清晰，不仅便于开发者维护，也使得机器(如搜索引擎、屏幕阅读器)能够更好地理解网页内容。

CSS 1. CSS选择器与@规则

CSS选择器用于精准选中HTML元素并施加样式，而@规则则用于定义CSS的整体行为、条件与元信息，二者共同构成样式表的核心机制。

CSS 2. 布局属性

CSS布局技术共同为页面元素提供位置控制、空间分配与排列对齐的核心解决方案，是构建视觉层级的基石。

CSS 3. 盒模型

在CSS中，每个元素都被表示为一个矩形的盒子。这个盒子由四个部分组成：内容(content)、内边距(padding)、边框(border)和外边距(margin)。

CSS 4. 外观和装饰

CSS外观和装饰为结构化内容(如HTML)添加视觉样式和布局，实现美化与界面设计。

CSS 5. 动画与交互

CSS通过过渡、变形和关键帧动画，为静态界面元素添加动态视觉效果和流畅的交互反馈，从而提升用户体验与页面活力。

CSS 6.其他功能属性

其他功能属性拓展了CSS的核心能力，通过变量、滤镜、混合模式、裁剪等属性，实现值的复用、视觉特效、高级合成与动态主题，增强了样式的灵活性、可维护性和表现力。

2. <!DOCTYPE html>

注意:必须放在第一行
用途:

1. 文档类型声明，告诉浏览器这是一个HTML5文档

2. 不是HTML标签，而是给浏览器的指令

3. 确保浏览器以标准模式渲染页面

3. <html> 根元素

<html> 元素是HTML文档的根元素，也称为根标签。它包含整个HTML文档的内容，以下是其常用属性和用法：

1. lang - 语言属性(最常用)

示例代码:

<html lang="zh-CN">

<!--

语言代码：

- zh-CN：简体中文(中国大陆)

- zh-TW：繁体中文(台湾)

- zh-HK：繁体中文(香港)

- en：英语

- ja：日语

-->

作用：

1. SEO优化：帮助搜索引擎识别页面语言

2. 可访问性：辅助屏幕阅读器选择正确的发音规则

3. 自动翻译：浏览器可以提示翻译或自动翻译

4. 字体渲染：某些浏览器会根据语言选择不同字体

5. 排版处理：影响连字符、断字规则等

6. 样式适配：可以通过CSS选择器根据语言应用不同样式

2. dir - 文本方向属性

示例代码:

<html dir="ltr">

<!-- 从左到右(默认，Left-to-Right) -->

<!-- 从右到左(Right-to-Left，用于阿拉伯语、希伯来语等) -->

<html dir="auto">

4. <head> 头部区域

<head> 元素是HTML文档的头部，包含文档的元数据(metadata)和资源链接。这些信息不会直接显示在页面上，但对页面的功能、表现和优化至关重要。

<!--

作用：

1. 确保浏览器正确解析文档字符

2. 防止中文等字符出现乱码

3. 推荐使用UTF-8，支持所有语言

-->

</head>

2 视口设置(移动端必备)

示例代码:

<head>

<!--

参数详解：

- width=device-width：宽度等于设备宽度

- initial-scale=1.0：初始缩放比例为1

- maximum-scale=1.0：最大缩放比例

- minimum-scale=1.0：最小缩放比例

- user-scalable=no：禁止用户缩放(慎用)

-->

</head>

3. 三种加载方式对比

示例代码:

<script src="blocking.js"></script>

<!-- async(异步，下载后立即执行) -->

<script src="async.js" async></script>

<!-- defer(延迟，DOM解析后执行，按顺序) -->

<script src="defer.js" defer></script>

<!--

加载策略对比：

1. 默认：阻塞HTML解析，下载并执行完才继续解析

2. async：异步下载，下载完成后立即执行(不保证顺序)

3. defer：异步下载，DOM解析完成后按顺序执行


推荐使用 defer，除非脚本不依赖DOM且相互独立才用async

-->

</head>

1. 基本选择器

基本选择器是CSS中最基础也是最常用的选择器类型。

1. 元素选择器

根据HTML元素名称选择元素。

/* 选择所有段落元素 */
p {
  color: blue;
}

2. 类选择器

根据class属性选择元素。

/* 选择所有class="highlight"的元素 */
.highlight {
  background-color: yellow;
}

3. ID选择器

根据id属性选择元素。

/* 选择id="header"的元素 */
#header {
  font-size: 2rem;
}

4. 通用选择器

选择所有元素。

/* 选择所有元素 */
* {
  margin: 0;
  padding: 0;
}

2. 组合选择器

组合选择器用于基于元素之间的关系选择元素。

1. 后代选择器

选择指定元素内的所有后代元素(不一定是直接子元素)。

/* 选择div内的所有p元素 */
div p {
  color: red;
}

2. 子选择器

选择指定元素的直接子元素。

/* 选择div的直接子元素p */
div > p {
  font-weight: bold;
}

3. 相邻兄弟选择器

选择紧接在另一元素后的元素，且二者有相同父元素。

/* 选择紧接在h2后的p元素 */
h2 + p {
  margin-top: 0;
}

4. 通用兄弟选择器

选择指定元素之后的所有兄弟元素。

/* 选择h2之后的所有p兄弟元素 */
h2 ~ p {
  color: green;
}

3. 属性选择器

属性选择器根据元素的属性及属性值来选择元素。

1. [attribute] 选择器

选择带有指定属性的元素。

/* 选择所有带有target属性的元素 */
[target] {
  border: 1px solid red;
}

2. [attribute=value] 选择器

选择带有指定属性和值的元素。

/* 选择所有type="text"的input元素 */
input[type="text"] {
  width: 200px;
}

3. [attribute~=value] 选择器

选择属性值中包含指定词汇的元素。

/* 选择title属性包含"flower"的元素 */
[title~=flower] {
  border: 1px solid green;
}

4. [attribute|=value] 选择器

选择属性值以指定值开头或以该值后跟连字符的元素。

/* 选择lang属性值以"en"开头的元素 */
[lang|=en] {
  color: blue;
}

5. [attribute^=value] 选择器

选择属性值以指定值开头的元素。

/* 选择href属性值以"https"开头的元素 */
a[href^="https"] {
  color: green;
}

6. [attribute$=value] 选择器

选择属性值以指定值结尾的元素。

/* 选择href属性值以".pdf"结尾的元素 */
a[href$=".pdf"] {
  background-color: yellow;
}

7. [attribute*=value] 选择器

选择属性值中包含指定值的元素。

/* 选择href属性值中包含"example"的元素 */
a[href*="example"] {
  font-style: italic;
}

4. 伪类选择器

伪类选择器用于定义元素的特殊状态。

1. 链接伪类

:link, :visited, :hover, :active

/* 未访问的链接 */
a:link {
  color: blue;
}

/* 已访问的链接 */
a:visited {
  color: purple;
}

/* 鼠标悬停时的链接 */
a:hover {
  color: red;
}

/* 活动链接(正在点击的链接) */
a:active {
  color: green;
}

2. 表单伪类

:focus, :checked, :disabled, :required

/* 获得焦点的输入框 */
input:focus {
  border-color: blue;
  box-shadow: 0 0 5px rgba(0, 0, 255, 0.5);
}

/* 被选中的复选框 */
input:checked {
  background-color: blue;
}

3. 结构伪类

:first-child, :last-child, :nth-child(), :nth-of-type()

/* 选择第一个子元素 */
p:first-child {
  font-weight: bold;
}

/* 选择第三个子元素 */
p:nth-child(3) {
  color: red;
}

/* 选择奇数行 */
tr:nth-child(odd) {
  background-color: #f2f2f2;
}

5. 伪元素选择器

伪元素选择器用于选择元素的特定部分。

1. ::before 和 ::after

在元素内容之前或之后插入内容。

/* 在p元素前添加内容 */
p::before {
  content: "→ ";
  color: blue;
}

/* 在p元素后添加内容 */
p::after {
  content: "!";
  font-weight: bold;
}

2. ::first-line

选择元素的第一行文本。

/* 选择p元素的第一行 */
p::first-line {
  font-weight: bold;
  color: blue;
}

3. ::first-letter

选择元素的第一个字母。

/* 选择p元素的第一个字母 */
p::first-letter {
  font-size: 2em;
  float: left;
  margin-right: 5px;
}

4. ::selection

选择用户选中的文本部分。

/* 设置文本选中时的样式 */
::selection {
  background-color: yellow;
  color: black;
}

6. @规则

@规则是CSS中的特殊指令，用于定义样式表的行为和结构。

1. @charset 字符集声明

定义样式表文件所使用的字符编码，必须在文件最前面声明(且前面不能有任何字符)。

/* 指定样式表使用UTF-8编码 */
@charset "UTF-8"

2. @import 导入规则

用于在当前样式表中导入另一个外部CSS文件。可放在样式表开头或@规则之后，其他规则之前。

/* 导入另一个CSS文件 */
@import url("theme.css")
/* 可附加媒体查询条件 */
@import "print.css" print

注意：影响加载性能，现代开发中更推荐使用构建工具或HTML的<link>标签合并文件。

3. @namespace 命名空间

定义XML命名空间，用于在包含多种XML语言(如XHTML、SVG)的文档中避免选择器冲突。

/* 声明XHTML和SVG的命名空间 */
@namespace url("http://www.w3.org/1999/xhtml");
@namespace svg url("http://www.w3.org/2000/svg");

/* 使用命名空间限定选择器 */
svg|a {
  stroke: blue;
}

注意：主要用于XHTML或混合XML文档，普通HTML开发中较少使用。

4. @media 媒体查询

根据设备或浏览器的特性(如视口宽度、屏幕方向、分辨率等)有条件地应用一组样式规则，是实现响应式设计的核心。

@media (max-width: 768px) {
  .menu {
    flex-direction: column;
  }
}

5. @supports 特性查询

检测浏览器是否支持某个特定的CSS属性或属性值，根据检测结果应用样式，常用于渐进增强。

@supports (display: grid) {
  .container {
    display: grid;
  }
}

6. @font-face 自定义字体

允许开发者指定在线字体供网页使用，突破“Web安全字体”的限制。

@font-face {
  font-family: 'MyWebFont';
  src: url('font.woff2') format('woff2');
  font-weight: 700;
}

7. @keyframes 关键帧动画

定义动画序列中的关键状态(帧)，然后通过animation属性将动画应用到元素上。

/* 定义动画 */
@keyframes colorChange {
  0% {
    background-color: red;
  }
  50% {
    background-color: blue;
  }
  100% {
    background-color: red;
  }
}

/* 应用动画 */
.animated-box {
  animation: colorChange 3s infinite;
}

注意：关键帧用百分比或关键词(from/to)定义，可控制动画的中间步骤。

8. @page 分页媒体规则

用于设置打印文档时的页面样式，如尺寸、边距和方向。

@page {
  size: A4 portrait;
  margin: 2cm;
}

注意：可以使用:first,:left,:right等伪类选择器为首页、左页、右页设置不同样式。

9. @layer 级联层

(CSS Cascades and Inheritance Level 5引入)用于显式定义样式规则的层级顺序，以更精细地控制样式层叠(cascade)。

/* 定义层级的顺序 */
@layer base, theme, utilities;

/* 在指定层中添加样式 */
@layer utilities {
  .mt-4 {
    margin-top: 1rem;
  }
}

注意：后声明的层比先声明的层优先级低(与层内样式特殊性无关)，用于管理大型项目中的样式优先级冲突。

10. @counter-style 自定义计数器样式

允许定义列表项计数器(list-item counter)的自定义显示样式，超出预定义样式(如decimal, disc)的限制。

@counter-style triangle {
  system: cyclic;
  symbols: "▶";
  suffix: " ";
}

注意：可以创建独特的列表标记，如使用特殊符号、字母序列等。

11. @property 自定义属性注册

(CSS Properties and Values API Level 1引入)用于显式注册一个自定义属性(CSS变量)，并定义其语法、继承行为及初始值，使其支持类型检查和动画。

@property --main-color {
  syntax: '';
  inherits: true;
  initial-value: #000;
}

注意：注册后，自定义属性可以被transition和animation动画化，并且浏览器能进行类型检查。

12. @scope 作用域规则

(实验性功能)用于将一组样式规则的作用范围限定在某个DOM子树内，防止样式泄露到外部，也防止外部样式侵入该范围。

@scope (.component) {
  .title {
    color: red;
  }
}

4. 网格布局(Grid)
