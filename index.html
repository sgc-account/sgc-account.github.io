代码类型

  • HTML
    HTML 是网页的结构和内容，它定义了页面上有什么(如标题、段落、图片)
  • CSS
    CSS 是网页的样式和外观，它控制内容如何被呈现(如颜色、布局、字体)
  • JavaScript
    JavaScript 是网页的行为和交互，它使页面能够响应用户操作并动态变化
