CSS 1. CSS选择器与@规则

CSS选择器用于精准选中HTML元素并施加样式，而@规则则用于定义CSS的整体行为、条件与元信息，二者共同构成样式表的核心机制。

CSS 2. 布局属性

CSS布局技术共同为页面元素提供位置控制、空间分配与排列对齐的核心解决方案，是构建视觉层级的基石。

CSS 3. 盒模型

在CSS中，每个元素都被表示为一个矩形的盒子。这个盒子由四个部分组成：内容(content)、内边距(padding)、边框(border)和外边距(margin)。

CSS 4. 外观和装饰

CSS外观和装饰为结构化内容(如HTML)添加视觉样式和布局，实现美化与界面设计。

CSS 5. 动画与交互

CSS通过过渡、变形和关键帧动画，为静态界面元素添加动态视觉效果和流畅的交互反馈，从而提升用户体验与页面活力。

CSS 6.其他功能属性

其他功能属性拓展了CSS的核心能力，通过变量、滤镜、混合模式、裁剪等属性，实现值的复用、视觉特效、高级合成与动态主题，增强了样式的灵活性、可维护性和表现力。

1. 基本选择器

基本选择器是CSS中最基础也是最常用的选择器类型。

1. 元素选择器

根据HTML元素名称选择元素。

/* 选择所有段落元素 */
p {
  color: blue;
}

2. 类选择器

根据class属性选择元素。

/* 选择所有class="highlight"的元素 */
.highlight {
  background-color: yellow;
}

3. ID选择器

根据id属性选择元素。

/* 选择id="header"的元素 */
#header {
  font-size: 2rem;
}

4. 通用选择器

选择所有元素。

/* 选择所有元素 */
* {
  margin: 0;
  padding: 0;
}

2. 组合选择器

组合选择器用于基于元素之间的关系选择元素。

1. 后代选择器

选择指定元素内的所有后代元素(不一定是直接子元素)。

/* 选择div内的所有p元素 */
div p {
  color: red;
}

2. 子选择器

选择指定元素的直接子元素。

/* 选择div的直接子元素p */
div > p {
  font-weight: bold;
}

3. 相邻兄弟选择器

选择紧接在另一元素后的元素，且二者有相同父元素。

/* 选择紧接在h2后的p元素 */
h2 + p {
  margin-top: 0;
}

4. 通用兄弟选择器

选择指定元素之后的所有兄弟元素。

/* 选择h2之后的所有p兄弟元素 */
h2 ~ p {
  color: green;
}

3. 属性选择器

属性选择器根据元素的属性及属性值来选择元素。

1. [attribute] 选择器

选择带有指定属性的元素。

/* 选择所有带有target属性的元素 */
[target] {
  border: 1px solid red;
}

2. [attribute=value] 选择器

选择带有指定属性和值的元素。

/* 选择所有type="text"的input元素 */
input[type="text"] {
  width: 200px;
}

3. [attribute~=value] 选择器

选择属性值中包含指定词汇的元素。

/* 选择title属性包含"flower"的元素 */
[title~=flower] {
  border: 1px solid green;
}

4. [attribute|=value] 选择器

选择属性值以指定值开头或以该值后跟连字符的元素。

/* 选择lang属性值以"en"开头的元素 */
[lang|=en] {
  color: blue;
}

5. [attribute^=value] 选择器

选择属性值以指定值开头的元素。

/* 选择href属性值以"https"开头的元素 */
a[href^="https"] {
  color: green;
}

6. [attribute$=value] 选择器

选择属性值以指定值结尾的元素。

/* 选择href属性值以".pdf"结尾的元素 */
a[href$=".pdf"] {
  background-color: yellow;
}

7. [attribute*=value] 选择器

选择属性值中包含指定值的元素。

/* 选择href属性值中包含"example"的元素 */
a[href*="example"] {
  font-style: italic;
}

4. 伪类选择器

伪类选择器用于定义元素的特殊状态。

1. 链接伪类

:link, :visited, :hover, :active

/* 未访问的链接 */
a:link {
  color: blue;
}

/* 已访问的链接 */
a:visited {
  color: purple;
}

/* 鼠标悬停时的链接 */
a:hover {
  color: red;
}

/* 活动链接(正在点击的链接) */
a:active {
  color: green;
}

2. 表单伪类

:focus, :checked, :disabled, :required

/* 获得焦点的输入框 */
input:focus {
  border-color: blue;
  box-shadow: 0 0 5px rgba(0, 0, 255, 0.5);
}

/* 被选中的复选框 */
input:checked {
  background-color: blue;
}

3. 结构伪类

:first-child, :last-child, :nth-child(), :nth-of-type()

/* 选择第一个子元素 */
p:first-child {
  font-weight: bold;
}

/* 选择第三个子元素 */
p:nth-child(3) {
  color: red;
}

/* 选择奇数行 */
tr:nth-child(odd) {
  background-color: #f2f2f2;
}

5. 伪元素选择器

伪元素选择器用于选择元素的特定部分。

1. ::before 和 ::after

在元素内容之前或之后插入内容。

/* 在p元素前添加内容 */
p::before {
  content: "→ ";
  color: blue;
}

/* 在p元素后添加内容 */
p::after {
  content: "!";
  font-weight: bold;
}

2. ::first-line

选择元素的第一行文本。

/* 选择p元素的第一行 */
p::first-line {
  font-weight: bold;
  color: blue;
}

3. ::first-letter

选择元素的第一个字母。

/* 选择p元素的第一个字母 */
p::first-letter {
  font-size: 2em;
  float: left;
  margin-right: 5px;
}

4. ::selection

选择用户选中的文本部分。

/* 设置文本选中时的样式 */
::selection {
  background-color: yellow;
  color: black;
}

6. @规则

@规则是CSS中的特殊指令，用于定义样式表的行为和结构。

1. @charset 字符集声明

定义样式表文件所使用的字符编码，必须在文件最前面声明(且前面不能有任何字符)。

/* 指定样式表使用UTF-8编码 */
@charset "UTF-8"

2. @import 导入规则

用于在当前样式表中导入另一个外部CSS文件。可放在样式表开头或@规则之后，其他规则之前。

/* 导入另一个CSS文件 */
@import url("theme.css")
/* 可附加媒体查询条件 */
@import "print.css" print

注意：影响加载性能，现代开发中更推荐使用构建工具或HTML的<link>标签合并文件。

3. @namespace 命名空间

定义XML命名空间，用于在包含多种XML语言(如XHTML、SVG)的文档中避免选择器冲突。

/* 声明XHTML和SVG的命名空间 */
@namespace url("http://www.w3.org/1999/xhtml");
@namespace svg url("http://www.w3.org/2000/svg");

/* 使用命名空间限定选择器 */
svg|a {
  stroke: blue;
}

注意：主要用于XHTML或混合XML文档，普通HTML开发中较少使用。

4. @media 媒体查询

根据设备或浏览器的特性(如视口宽度、屏幕方向、分辨率等)有条件地应用一组样式规则，是实现响应式设计的核心。

@media (max-width: 768px) {
  .menu {
    flex-direction: column;
  }
}

5. @supports 特性查询

检测浏览器是否支持某个特定的CSS属性或属性值，根据检测结果应用样式，常用于渐进增强。

@supports (display: grid) {
  .container {
    display: grid;
  }
}

6. @font-face 自定义字体

允许开发者指定在线字体供网页使用，突破“Web安全字体”的限制。

@font-face {
  font-family: 'MyWebFont';
  src: url('font.woff2') format('woff2');
  font-weight: 700;
}

7. @keyframes 关键帧动画

定义动画序列中的关键状态(帧)，然后通过animation属性将动画应用到元素上。

/* 定义动画 */
@keyframes colorChange {
  0% {
    background-color: red;
  }
  50% {
    background-color: blue;
  }
  100% {
    background-color: red;
  }
}

/* 应用动画 */
.animated-box {
  animation: colorChange 3s infinite;
}

注意：关键帧用百分比或关键词(from/to)定义，可控制动画的中间步骤。

8. @page 分页媒体规则

用于设置打印文档时的页面样式，如尺寸、边距和方向。

@page {
  size: A4 portrait;
  margin: 2cm;
}

注意：可以使用:first,:left,:right等伪类选择器为首页、左页、右页设置不同样式。

9. @layer 级联层

(CSS Cascades and Inheritance Level 5引入)用于显式定义样式规则的层级顺序，以更精细地控制样式层叠(cascade)。

/* 定义层级的顺序 */
@layer base, theme, utilities;

/* 在指定层中添加样式 */
@layer utilities {
  .mt-4 {
    margin-top: 1rem;
  }
}

注意：后声明的层比先声明的层优先级低(与层内样式特殊性无关)，用于管理大型项目中的样式优先级冲突。

10. @counter-style 自定义计数器样式

允许定义列表项计数器(list-item counter)的自定义显示样式，超出预定义样式(如decimal, disc)的限制。

@counter-style triangle {
  system: cyclic;
  symbols: "▶";
  suffix: " ";
}

注意：可以创建独特的列表标记，如使用特殊符号、字母序列等。

11. @property 自定义属性注册

(CSS Properties and Values API Level 1引入)用于显式注册一个自定义属性(CSS变量)，并定义其语法、继承行为及初始值，使其支持类型检查和动画。

@property --main-color {
  syntax: '';
  inherits: true;
  initial-value: #000;
}

注意：注册后，自定义属性可以被transition和animation动画化，并且浏览器能进行类型检查。

12. @scope 作用域规则

(实验性功能)用于将一组样式规则的作用范围限定在某个DOM子树内，防止样式泄露到外部，也防止外部样式侵入该范围。

@scope (.component) {
  .title {
    color: red;
  }
}

3. 弹性布局(Flexbox)

Flexbox 是一种一维布局模型，非常适合处理行或列方向上的元素排列。

容器属性(父元素)：

/* 容器启用Flex */
display: flex; /* 或 inline-flex */
flex-direction: row; /* 主轴方向：row(默认)| row-reverse | column | column-reverse */
flex-wrap: nowrap; /* 是否换行：nowrap(默认)| wrap | wrap-reverse */

/* 主轴对齐方式(水平方向默认) */
justify-content: flex-start; /* start | end | center | space-between | space-around | space-evenly */

/* 交叉轴对齐方式(垂直方向默认) */
align-items: stretch; /* stretch(默认) | flex-start | flex-end | center | baseline */

/* 多行时的交叉轴对齐(需有wrap) */
align-content: stretch; /* stretch(默认) | flex-start | flex-end | center | space-between | space-around */

/* 复合属性 */
flex-flow: row nowrap; /* flex-direction 和 flex-wrap 的简写 */

项目属性(子元素)：

/* 项目排序(默认0，可为负值) */
order: 0;

/* 放大比例(默认0，不放大) */
flex-grow: 0;

/* 缩小比例(默认1，空间不足时缩小) */
flex-shrink: 1;

/* 项目初始大小 */
flex-basis: auto; /* auto | <length> | content */

/* 复合属性(最常用) */
flex: 1; /* 等同于 flex: 1 1 0% */
flex: auto; /* 等同于 flex: 1 1 auto */
flex: none; /* 等同于 flex: 0 0 auto */
flex: 0 0 200px; /* 固定宽度200px，不放大不缩小 */

/* 单独对齐(覆盖容器的align-items) */
align-self: auto; /* auto | flex-start | flex-end | center | baseline | stretch */

4. 网格布局(Grid)

Grid 是一种二维布局系统，可以同时控制行和列，非常适合整体页面布局。

容器属性(父元素)：

/* 容器启用Grid */
display: grid; /* 或 inline-grid */

/* 定义网格列(最常用) */
grid-template-columns: 1fr 2fr 1fr; /* 三列，中间列是两侧的两倍 */
grid-template-columns: repeat(3, 1fr); /* 重复3列，每列1fr */
grid-template-columns: 200px 1fr auto; /* 第一列固定，第二列自适应，第三列根据内容 */
grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); /* 响应式网格 */

/* 定义网格行 */
grid-template-rows: 100px auto 150px; /* 三行，中间行自适应 */

/* 网格区域命名(用于子元素定位) */
grid-template-areas:

"header header header"

"sidebar main main"

"footer footer footer";


/* 行/列间距 */
gap: 20px; /* 行和列间距都为20px */
row-gap: 10px; /* 行间距 */
column-gap: 15px; /* 列间距 */

/* 项目在单元格内的对齐 */
justify-items: stretch; /* 水平：start | end | center | stretch(默认) */
align-items: stretch; /* 垂直：start | end | center | stretch(默认) */
place-items: center; /* 简写：align-items justify-items */

/* 整个网格在容器内的对齐(网格总尺寸小于容器时) */
justify-content: start; /* start | end | center | stretch | space-between | space-around | space-evenly */
align-content: start; /* 同上 */

/* 自动生成网格轨道大小(未定义的行/列) */
grid-auto-rows: 100px; /* 自动生成的行高100px */
grid-auto-columns: 1fr; /* 自动生成的列宽1fr */
grid-auto-flow: row; /* 自动放置方向：row(默认)| column | row dense | column dense */

项目属性(子元素)：

/* 指定项目位置(行/列起止) */
grid-column-start: 1;
grid-column-end: 3; /* 跨越第1-2列(不包括3) */
grid-row-start: 1;
grid-row-end: 2;

/* 简写形式(更常用) */
grid-column: 1 / 3; /* 等同于上面两行 */
grid-column: 1 / span 2; /* 从第1列开始，跨越2列 */
grid-row: 2 / 4;

/* 使用命名区域定位 */
grid-area: header; /* 放置在名为header的区域 */

/* 单独对齐(覆盖容器的align-items/justify-items) */
justify-self: stretch; /* start | end | center | stretch(默认) */
align-self: center; /* start | end | center | stretch(默认) */
place-self: center; /* 简写：align-self justify-self */

6. 多列布局(Multi-column)

常用代码：

/* 列数 */
column-count: 3; /* 固定列数 */

/* 列宽(与column-count互斥，浏览器自动计算列数) */
column-width: 200px; /* 每列最小宽度200px */

/* 简写(最常用) */
columns: 3 250px; /* 建议3列，每列最小250px */

/* 列间距 */
column-gap: 30px; /* 列与列之间的间距 */

/* 列分隔线 */
column-rule: 1px solid #ccc; /* 简写：宽度 样式 颜色 */
column-rule-width: 1px;
column-rule-style: solid;
column-rule-color: #ccc;

/* 跨列元素 */
column-span: none; /* 不跨列(默认) */
column-span: all; /* 横跨所有列 */
Multi-column 示例

CSS的多列布局允许我们轻松创建多列文本布局，而无需使用复杂的浮动或定位技术。这种布局特别适合新闻文章、博客内容等长篇文本。

通过使用column-count属性，可以指定内容应该分成多少列。column-gap属性控制列之间的间距，而column-rule属性可以在列之间添加分隔线。

此外，还可以使用column-width属性指定每列的理想宽度，浏览器会根据可用空间自动调整列数。

多列布局还支持跨列元素，使用column-span属性可以让元素跨越所有列，非常适合标题或重要引用的展示。

2. 内容区域(Content)

内容区域包含元素的真实内容，如文本、图像等。可以使用 width 和 height 属性设置内容区域的尺寸。

常用代码：

/* 内容尺寸属性 */
width: 200px; /* 设置内容宽度 */
height: 150px; /* 设置内容高度 */
min-width: 100px; /* 设置最小宽度 */
max-width: 500px; /* 设置最大宽度 */
min-height: 50px; /* 设置最小高度 */
max-height: 300px; /* 设置最大高度 */

/* 常用单位 */

6. Box Sizing 属性

/* border-box: 总宽度 = width (已经包含padding和border) */

4. 媒体查询 (Media Queries)

媒体查询允许您根据设备特性(如屏幕尺寸、方向等)应用不同的样式。

常用代码：

/* 媒体查询语法 */
@media media-type and (media-feature) {
/* CSS规则 */
}

/* 媒体类型 */
@media screen { ... } /* 屏幕设备 */
@media print { ... } /* 打印设备 */
@media all { ... } /* 所有设备 */

/* 响应式断点示例 */
@media (max-width: 768px) { ... } /* 最大宽度 */
@media (min-width: 769px) { ... } /* 最小宽度 */
@media (orientation: portrait) { ... } /* 竖屏 */
@media (orientation: landscape) { ... } /* 横屏 */
@media (prefers-color-scheme: dark) { ... } /* 深色模式偏好 */
@media (prefers-reduced-motion: reduce) { ... } /* 减少动画偏好 */

/* 响应式断点示例 */
/* 手机 */
@media (max-width: 576px) {
  .container {
    padding: 10px;
  }
}

/* 平板 */
@media (min-width: 577px) and (max-width: 768px) {
  .container {
    padding: 15px;
  }
}

/* 桌面 */
@media (min-width: 769px) and (max-width: 1024px) {
  .container {
    padding: 20px;
  }
}

/* 大桌面 */
@media (min-width: 1025px) {
  .container {
    padding: 25px;
  }
}

/* 打印样式 */
@media print {
  .no-print {
    display: none;
  }
  body {
    font-size: 12pt;
    line-height: 1.4;
  }
}

/* 深色模式 */
@media (prefers-color-scheme: dark) {
  :root {
    --bg-color: #1a1a1a;
    --text-color: #ffffff;
  }
}

5. 计数器 (Counters)

CSS计数器允许您自动对元素进行编号，类似于有序列表但更加灵活。

常用代码：

/* 计数器属性 */
counter-reset: section; /* 重置或创建计数器 */
counter-increment: section; /* 递增计数器 */

/* 使用计数器 */
counter: counter(section); /* 显示计数器值 */
counter: counters(section, "."); /* 显示嵌套计数器 */

ol {
  counter-reset: section; /* 创建名为section的计数器 */
  list-style-type: none;
}

li {
  counter-increment: section; /* 每个li元素递增计数器 */
}

li::before {
  content: counters(section, ".") " "; /* 显示嵌套计数器 */
}

/* 自定义计数器样式 */
counter: counter(section, lower-roman); /* 小写罗马数字 */
counter: counter(section, upper-alpha); /* 大写字母 */
counter: counter(section, decimal-leading-zero); /* 前导零十进制 */

body {
  counter-reset: chapter figure;
}

.chapter {
  counter-increment: chapter;
  counter-reset: figure;
}

.chapter::before {
  counter: "Chapter " counter(chapter) ": ";
}

.figure {
  counter-increment: figure;
}

.figure::before {
  counter: "Figure " counter(chapter) "." counter(figure);
}

6. 其他功能属性

这些属性提供了额外的功能和效果。

常用代码：

/* 用户选择 */
user-select: auto; /* 浏览器默认行为 */

user-select: text; /* 文本可选择 */
user-select: all; /* 点击选择全部文本 */

/* 指针事件 */
pointer-events: auto; /* 元素响应指针事件 */
pointer-events: none; /* 元素不响应指针事件 */

scroll-behavior: auto; /* 立即跳转 */
scroll-behavior: smooth; /* 平滑滚动 */

/* 文字渲染 */
text-rendering: auto; /* 浏览器决定 */
text-rendering: optimizeSpeed; /* 优先速度 */
text-rendering: optimizeLegibility; /* 优先可读性 */
text-rendering: geometricPrecision; /* 几何精度 */

/* 图像渲染 */
image-rendering: auto; /* 浏览器决定 */
image-rendering: crisp-edges; /* 保留对比度和边缘 */
image-rendering: pixelated; /* 像素化缩放 */

/* 书写模式 */
writing-mode: horizontal-tb; /* 水平从上到下 */
writing-mode: vertical-rl; /* 垂直从右到左 */
writing-mode: vertical-lr; /* 垂直从左到右 */

/* 文本方向 */
direction: ltr; /* 从左到右 */
direction: rtl; /* 从右到左 */
